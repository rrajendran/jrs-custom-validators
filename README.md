##Adding custom JSR-303 Validators


##@ValidDate

Validates if the give string has a valid date or not. Can provide date format also. Default is yyyy-MM-dd
<br />
Example
<br />
##@ValidDate(format="dd-MM-yyyy")
<br />
String date;
